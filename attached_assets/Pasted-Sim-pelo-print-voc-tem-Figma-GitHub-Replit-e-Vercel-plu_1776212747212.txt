Bora. **Spec v0.1 — Living Software Canvas**

## 1. Ideia central

Um canvas visual para projetos Next.js onde cada nó representa uma parte real do sistema:

**página, rota, componente, API, auth, proxy/middleware, banco, gateway, botão/link.**

Cada nó também funciona como um **mini-chat contextual**, com prompt, anexos, prints, regras, bugs e tarefas.

## 2. MVP realista

Primeira versão não altera código sozinha. Ela faz:

1. Canvas com nós e conexões.
2. Chat lateral por nó.
3. Upload de anexos.
4. Snapshot do estado atual.
5. Comparação entre snapshots.
6. Exportação JSON.
7. Prompt consolidado para agente IA.

## 3. Stack

* **Next.js App Router**
* **TypeScript**
* **xyflow/React Flow**
* **Vercel AI SDK/Gateway**
* **Vercel Blob** para anexos
* **GitHub** para versionar snapshots
* **Graph-It-Live** depois, como scanner do projeto real

## 4. Modelo de dados base

```ts
type CanvasNode = {
  id: string
  type: 'page' | 'route' | 'component' | 'api' | 'auth' | 'proxy' | 'database' | 'gateway' | 'ui-action'
  label: string
  filePath?: string
  routePath?: string
  description?: string
  prompts: string[]
  notes: string[]
  attachments: Attachment[]
  status: 'draft' | 'mapped' | 'needs-review' | 'approved'
}

type CanvasEdge = {
  id: string
  source: string
  target: string
  type: 'navigates-to' | 'renders' | 'calls-api' | 'redirects' | 'auth-guards' | 'queries-db'
  label?: string
}

type Snapshot = {
  id: string
  name: string
  createdAt: string
  nodes: CanvasNode[]
  edges: CanvasEdge[]
}
```

## 5. Telas

**Canvas principal**
Mapa visual estilo n8n.

**Painel lateral do nó**
Chat, descrição, anexos, prompt, arquivo vinculado, rota vinculada.

**Snapshots**
Lista de versões salvas tipo commit visual.

**Diff**
Mostra o que mudou: nó criado, edge removida, prompt alterado.

## 6. Regra de segurança

Nada de “aplicar mudança no código” no MVP.
Primeiro: **gerar plano/patch sugerido**.

Aplicação automática só na v0.2 ou v0.3. Senão vira motosserra com Wi-Fi.
